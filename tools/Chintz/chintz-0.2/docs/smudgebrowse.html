Smudge: The BrowseButton Tag

BrowseButtons can only be used in Lines, and must be placed immediately before the < /Line> closing tag. When clicked the button (which is labelled with three dots ...) pops up a file browser window, which allows the chintz user to select a filename to be place in the Line's text box, and assigned to the variable defined by the Line's id. See the Line page for an example.
