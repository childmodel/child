Smudge: The Switch Tag

Switch applies only to CheckBoxes and RadioButtons. It allows the buttons to control the enabling of Frames and Boxes with a pre-existing name, given by the Frame or Box id. Switch takes two parameters, the name of the Frame or Box and true or false. If the second paramter is set true, then the Frame is only enabled when the RadioButton or CheckBox is checked, with the opposite behaviour for false. The form of the tag is
< Switch> Name true/false < /Switch>
