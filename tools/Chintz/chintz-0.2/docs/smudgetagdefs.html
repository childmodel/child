Smudge: Tag Definitions

There are 17 defined tags in all. Not all combinations of tags are allowed. Attempting to use tags in unrecognised places will most likely end in them being ignored. So, for example, the < Browsebutton> tag can only be used within a < Line> < /Line>. Note the capitalization of the tags - this is important for them to be recognised.
  • Containers
    • < Page>< /Page>
    • < Frame>< /Frame>
    • < Box>< /Box>
    • < Buttons>< /Buttons>
  • Text Entry
    • < Line>< /Line>
    • < Biline>< /Biline>
  • Clickable items
    • < CheckBox>< /CheckBox>
    • < RadioButton>< /RadioButton>
    • < BrowseButton>
  • Modifiers
    • < Title>< /Title>
    • < Style>< /Style>
    • < Bold>< /Bold>
    • < Enabled>< /Enabled>
    • < Geometry>< /Geometry>
    • < Origin>< /Origin>
    • < Switch>< /Switch>
    • < Checked>< /Checked>
