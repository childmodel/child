Smudge: The Box Tag

As with Frames, Boxes provide a rectangular switchable container. The main difference is that the Box title is inset into the boundary, and can be made to stand out somewhat using the < Bold> modifier. Box titles are drawn black. Example:-

< Page>
< Title> Setup < /Title>
    < Frame,id=Firstframe>
      < Style> NoFrame < /Style>
      < Geometry> 10 10 100 200 < /Geometry>
      < Enabled> false < /Enabled>
      < Title> Mesh < /Title>

      < Box,id=Firstbox>
        < Geometry> 10 10 70 100 < /Geometry>
        < Title> Slope < /Title>
        < Bold> true < /Bold>
      < /Box>

    < /Frame>
< /Page>

a Page, labelled "Setup", containing a frame labelled "Mesh", which in turn contains a box labelled "Slope". The title is set to print in boldface. The Box geometry is relative to the Frame which contains it, so its origin is 10 pixels into the frame. The id "Firstbox" can be used for switching. By default the box is enabled, but since it is contained within a disabled frame, it will not be acessible until the Frame is switched on.
