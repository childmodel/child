Smudge: The Frame tag

Frames provide a rectangular area, with or without a border, within which other elements can be situated. Borders are manipulated using the < Style> modifier. Frames can be switched on or off by RadioButtons or Checkboxes, and so provide a means for grouping sets of elements which need to be enabled or disabled together. The id of the frame is used for the purpose of setting switches (see RadioButton and CheckBox). The frame can be set to be disabled initially by using the Enabled modifier (see example below: the default is for the frame to be enabled). The size of the frame is set by the < Geometry> modifier. If this modifier is omitted then the default size is one quarter page (i.e. 375 by 260). To set the size, include four values between the Geometry tags:

< Geometry> Xorigin Yorigin Xsize Ysize < /Geometry>

The values are measured from top left.

A title can be added, which will be drawn in blue italic within the body of the frame at top left. At the moment the space available for the title, and its style, cannot be modified.

Example:-

< Page>
< Title> Setup < /Title>
    < Frame,id=Firstframe>
      < Style> NoFrame < /Style>
      < Geometry> 10 10 100 200 < /Geometry>
      < Enabled> false < /Enabled>
      < Title> Mesh < /Title>
    < /Frame>
< /Page>

a page, labelled "Setup", containing a frame labelled "Mesh". The frame will not be visible to the user as the Style is set to NoFrame. The Geometry modifier lies between the < Frame > < /Frame> tags, and so sets the frame size (rather than, for example, the page size). The frame is 100 pixels wide and 200 pixels deep, starting 10 pixels into the page. The id "Firstframe" can be used to switch the frame between enabled and disabled. Initially the frame will be disabled (the chintz user will not be able to modify any of the elements contained within the frame.)
