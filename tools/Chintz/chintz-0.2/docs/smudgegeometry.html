Smudge: The Geometry Tag

< Geometry> allows the setting of the size and position of Frames, Boxes, Buttons, Lines(and Line Titles), CheckBoxes and RadioButtons. The tag takes four parameters, the Xorigin, the Yorigin, the Xsize and the Ysize. The origin gives the position relative to the container within which the current element is set. All values are given in pixels. The tag has the form
< Geometry> Xorigin Yorigin Xsize Ysize < /Geometry>
Note that the parameters are space delimited.
