Smudge: The Bold Tag

The < Bold> tag takes either a true or false value. It allows the text in Frames, Boxes, Buttons, Lines(and Line Titles), CheckBoxes and RadioButtons to be set to boldface or normal fonts. The form of the tag is
< Bold> true/false < /Bold>
