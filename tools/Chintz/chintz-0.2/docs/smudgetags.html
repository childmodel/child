Smudge: Tags

The basic tag within which all others must be contained is the < Page> tag. Each Page will appear as a new tab in the chintz user interface. No Page can contain futher nested Pages. Three futher types of container are available, Frames, Boxes and buttons. The arrangement of Frames and Boxes is hierarchical - all appearing between a < Page> tag and its closing < /Page> will be drawn on the same Page in the chintz user interface, but Frames may be nested within Frames or Boxes, and Boxes within Boxes or Frames

Thus:-

< Page,id=firstPage>
    < Frame>
        < line> < Title> A Text Line\nWith a line break < /Title> < /line>
    < /Frame>
< /Page>

< Page,id=second Page>
    < Frame>
        < Frame>
            < Title> A Nested Frame < /Title>
        < /Frame>
    < /Frame>
< Page>
defines two Pages, the first with a Frame drawn on it, within which there appears a text Box. Note the use of \n to break the text over two lines. The second Page contains two Frames, one drawn within the other, the second Frame bearing some title text. Indentation of the tags is included for clarity, but is not required. Similarly , blank lines will be ignored, and tags may appear on the same line or separate lines.
