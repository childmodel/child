Smudge: The Line Tag

A Line provides the second of four ways of setting variables for the child control file. Again, the id is used to provide the name of the child variable to be written out, but a text box is used to set its value. The Line provides a text box with a Title, each of which has a separate Geometry, so that their sizes can be set separately. The < Title> < /Title> tag can in this case only have a < Geometry> embedded within it to set its size. < Geometry> set outside of the Title, but within the < Line> < /Line> delimiters sets the size of the text box. The title and text box can also each separately be set to boldface. Again, a Line must be set into a separate Frame if it is to be switched on or off.
A further special tag is available only for the Line tag, namely < BrowseButton>. This tag must appear at the end of the specification, just before < /Line>. There is no closing < /BrowseButton>. This draws a small clickable button to the right of the textbox, which when clicked brings up a file browser. The text box value and the variable named in the Line id are set to the value of the file selected.

Hence:-

< Page>
< Title> Setup < /Title>
    < Frame,id=Firstframe>
      < Style> NoFrame < /Style>
      < Geometry> 10 10 100 200 < /Geometry>
      < Enabled> false < /Enabled>
      < Title> Mesh < /Title>

      < Line,id=NUM_PTS>
        < Geometry> 80 10 110 20 < /Geometry>
        < Title> Number of Random \nGrid Points
            < Geometry> 10 10 70 20 < /Geometry>
        < /Title>
        < BrowseButton>
      < /Line>

    < /Frame>
< /Page>

a page, labelled "Setup", containing a frame labelled "Mesh", which in turn contains a Line labelled "Number of Random Grid Points". This title will be Drawn to the left of the text box (starting 10 pixels into the Frame). The value typed into the text box will be assigned to the variable NUM_PTS, for writing to the control file, as in the Buttons example. Again if the id is omitted, then an internally generated variable name will be written to the control file.
