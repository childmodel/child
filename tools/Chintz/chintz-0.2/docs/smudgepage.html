Smudge: The Page tag

Pages are of fixed size : 750 across by 520 deep. This can currently only be changed by editing Chintz_base.cpp and smudge_exec.cpp and re-compiling. The page is the basic container for smudge, within which all other elements must be defined. A page can bear a title, which will appear in the chintz UI as labelling the tab on which the user clicks to bring the page into view.

So:-

< Page,id=firstpage>
< Title> Setup < /Title>
< /Page>

defines a single page, with a tab labelled "Setup". The id is not currently used by smudge, but can be useful for labelling the pages within the smudge.ui file.
