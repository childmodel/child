Smudge: The Enabled Tag

The Enabled Tag allows the initial state of Frames, Boxes, Buttons and Lines to be determined. If Enabled is set to false, then the chintz user will not be able to access these elements or any of their contents (they will appear "greyed out" in the UI.) until they are Enabled by a Switch. As for < Bold>, the form of the tag is
< Enabled> true/false < /Enabled>
